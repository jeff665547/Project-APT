CompStatExpressionProbeSetResultType

This structure provides storage for the comparison analysis results for a single probe set.

Properties

float ChangePValue - A p-value that indicates the significance level of the Change call.

float SignalLogRatio - The change in expression level for a transcript between a baseline and an experiment array. This change is expressed as the log2 ratio.

float SignalLogRatioLow - The lower limit of the Log Ratio within a 95% confidence range.

float SignalLogRatioHigh - The upper limit of the Signal Log Ratio within a 95% confidence range.

unsigned short NumCommonPairs - The intersection of the probe pairs from the baseline and experiment that are used by the Statistical Expression algorithm to make the Change call in a comparison analysis.

unsigned char Change - The call that indicates the change in transcript level between a baseline and an experiment array. See the constants below.  

Methods

_CompStatExpressionProbeSetResultType operator= - A copy operator.  

Constants

The following are constants defined to assist with the interpretation of the above structure.

EXP_UNKNOWN_COMP_CALL_TYPE - An unknown change call.

EXP_INCREASE_CALL_TYPE - An "increase" change call.

EXP_DECREASE_CALL_TYPE - A "decrease" change call.

EXP_INCREASE_MODERATE_CALL_TYPE - A "moderate increase" change call.

EXP_DECREASE_MODERATE_CALL_TYPE - A "moderate decrease" change call.

EXP_NO_CHANGE_CALL_TYPE - A "no change" chance call.

EXP_NO_COMP_CALL_TYPE - A "no call" change call.

const char *ExpressionCompCallString[] - String representation of the change call. Indexed by the "Change" property in the CompStatExpressionProbeSetResultType structure.
