CExpStatAlgSettings

This class provides storage for the algorithm parameters.

Enumerants

NormalizationOptionsEnum - Defines the type of normalization to perform.
NORM_TO_ALL_PROBE_SETS - Normalize to all probe sets on the array.
NORM_TO_SELECTED_PROBE_SETS - Normalize to a select set of probe sets.
DEFINED_NORMALIZATION_FACTOR - Use a defined normalization factor.
ScalingOptionsEnum - Defines the scaling to perform.
SCALE_TO_ALL_PROBE_SETS - Scale to all probe sets on the array.
SCALE_TO_SELECTED_PROBE_SETS - Scale to a select set of probe sets.
DEFINED_SCALING_FACTOR - Scale to a defined scaling factor.

Properties

NormalizationOptionsEnum NormMethod - Indicates which normalization method to use.

ScalingOptionsEnum SFMethod - Indicates which scaling method to use.

list<string> ScaleGenes - The list of probe sets to use when scaling. This is used when the SFMethod is set to SCALE_TO_SELECTED_PROBE_SETS.

list<string> NormGenes - The list of probe sets to use when normalizing. This is used when the NormMethod is set to NORM_TO_SELECTED_PROBE_SETS.

float NumberHorZones - The number of horizontal zones to use for calculating the background.

float NumberVertZones - The number of vertical zones to use for calculating the background.

float NumberBGCells - The percentage of features in each zone to use for calculating the background.

float NormFactor - The multiplier used to adjust the signal values for a comparison analysis.

float ScaleFactor - The multiplier used to adjust the signal value.

float TGT - The target signal value used for scaling.

float  IntensityLowPercent - The percentage of features (low intensity features) to exclude of those being used in the background calculation.

float IntensityHighPercent - The percentage of features (high intensity features) to exclude of those being used in the background calculation.

float Alpha1 - Significance level used to define a p-value cutoff for present detection calls.

float Alpha2 - Significance level used to define a p-value cutoff for absent detection calls.

float Tau - A small positive number that can be adjusted to increase of decrease sensitivity and/or specificity of the analysis.

float SaturatedIntensity - An intensity cutoff to determine the saturation point of the acquisition system (change calls).

float Epsilon - Tuning constant to avoid divide by zero in background preprocessing.

float ContrastTau - The tau value for the construction of ideal mismatches (see SADD). Used when the ideal MM approaches the value of the PM.

float TuningConstantCSB - The tuning constant for the Tukey Biweight rejection of outliers in computing Specific Background. Usually set to the standard value of 5 (5*MAD).

float TuningConstantCAvgLogInten - The tuning constant for the Tukey Biweight rejection of outliers when computing Signal.

float TuningConstantCAvgLogRatio - The tuning constant for the Tukey biweight rejection of outliers when computing Signal Log Ratio.

float TuningConstantCGammas - Tuning constant for Tukey biweight used in computing background for change calls.

float EpsilonSB - Constant used to division by zero variance when computing Specific Background.

float EpsilonAvgLogInten - Constant used to avoid division by zero variance values when computing Signal.

float EpsilonAvgLogRatio - Constant used to avoid division by zero variance when computing Signal-Log-Ratio.

float EpsilonGammas - Constant used to avoid dividing by zero variances when computing change calls

float SmoothFactorBG - Controls the smoothness of interpolation between zones for Background.

float NoiseFrac - Controls preprocessing Background subtraction (how close to zero to approach).

float ScaleTau - Controls the rate of approach of the ideal MM to PM. See formula in SADD.

float Delta - Smallest change in values allowed (Specific Background) between ideal MM and PM.

float Gamma1H - Significance level used to define a "Changed" p-value cutoff for change calls for high-range intensities.

float Gamma1L - Significance level used to define a "Changed" p-value cutoff for change calls for low intensities.

float Gamma2H - Significance level used to define a "Marginal" p-value cutoff for change calls for high intensities.

float Gamma2L - Significance level used to define a "Marginal" p-value cutoff for change calls for low intensities.

float Perturbation - The range by which the normalization factor is adjusted up or down for change calls.

float CMultiplier - This value scales PM-B values used in change calls to be comparable in magnitude to PM-MM values.

float BHCoef - High intensity for change call gamma interpolation is defined to be BHCoef * trimmed mean of PM on the chip(s).

float BLCoef - Low intensity for change call gamma interpolation is defined to be BLCoef * trimmed mean of PM on the chip(s)

float BiasCorrect - Multiplies the log signal by (1+BiasCorrect) to correct the log-log slope of signal-concentration towards 1.0.

float RelConfInterval - The desired coverage of the confidence interval for Signal Log Ratio.

float STP - The number of standard deviations away from the mean intensity difference. This value is used when computing the normalization factor.

Methods

None

Constructor/Destructor

CExpStatAlgSettings() - Sets the properties of the class to a default set of values.
