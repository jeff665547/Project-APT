CExpressionAlgorithmImplementation

This class provides the high level interface to run the algorithm.

Enumerants

None

Properties

None

Methods

void Clear() - Deallocates memory used by the class to store data.

string GetError() const - Returns an error string generated by the class when an error occured.

void SetLibPath(const char *libPath) - Sets the path to the GeneChip library files (CDF/PSI).

int GetNumResults() const - Returns the number of results (the number of probe sets on the array).

bool DoesCompDataExists() const - Flag indicating that comparison analysis results are available.

string GetProbeSetName(int index) const - Returns the name of the probe set at the 0 based "index" position.

AbsStatExpressionProbeSetResultType * GetAbsStatResult(int index) - Returns the absolute analysis results for the 0 based "index" probe set.

CompStatExpressionProbeSetResultType *GetCompStatResult(int index) - Returns the comparison analysis results for the 0 based "index" probe set. This function should not be called unless comparison results exist.

CExpStatAlgSettings &GetParameters() - Returns a reference to the algorithm parameters class.

bool RunStat(const char *celFile, const char *baseline) - Executes the analysis algorithm on the input "celFile" using the "baseline" file as the baseline CEL file. If the "baseline" argument is null or empty then only an absolute analysis is performed. If the return value is true then the analysis results will be available using the GetAbsStatResult() and GetCompStatResult() functions.

AllZonesInfoType & GetBackgroundZoneInfo() - Returns information about the background values per zone.

ControlInformationList & GetControlInfo() - Returns corner and checkerboard + and - information. The information is the average intensity for the control and the number of features for the control.

float GetRawQ() - Returns the RawQ value computed for the array (CEL file).

AvgStdvMinMaxType GetBgStats() - Returns the average, standard deviation, minimum and maximum background values.

AvgStdvMinMaxType GetNoiseStats() - Returns the average, standard deviation, minimum and maximum noise values.

CExpressionAlgorithmImplementation() - Initializes the class.

~CExpressionAlgorithmImplementation() - Deallocates memory used by the class.
