AbsStatExpressionProbeSetResultType

This structure provides storage for the absolute analysis results for a single probe set.

Properties

float DetectionPValue - A p-value that indicates the significance level of the Detection call.

float Signal - A measure of the abundance of a transcript.

unsigned short NumPairs - The number of probe pairs in the probe set.

unsigned short NumUsedPairs - (NumPairs - #masked probe pairs - #saturated MM probe pairs). This is the number of probe pairs used by the algorithm to make the detection call.

unsigned char Detection - The call in an absolute analysis that indicates if the transcript was present, absent, marginal or no call. See the constants below.

Methods

_AbsStatExpressionProbeSetResultType operator= - A copy operator.

Constants

The following are constants defined to assist with the interpretation of the above structure.

EXP_PRESENT_CALL_TYPE - A "present" call.

EXP_MARGINAL_CALL_TYPE - A "marginal" call.

EXP_ABSENT_CALL_TYPE - An "absent" call.

EXP_NO_ABS_CALL_TYPE - A "no call" call.

const char *ExpressionAbsCallString[] - String representation of the change call. Indexed by the "Detection" property in the AbsStatExpressionProbeSetResultType structure.
